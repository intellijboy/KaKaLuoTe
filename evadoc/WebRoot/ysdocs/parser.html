Parser（解析器）

使用方法

  1. $.parser.parse();       // 解析所有页面
  2. $.parser.parse('#cc');  // 解析指定节点
	$.parser.parse();             // 解析所有页面
	$.parser.parse('#cc');        // 解析指定节点

 

属性

属性名 属性值类型 描述 默认值
$.parser.auto boolean 定义是否自动解析EasyUI组件。 true

 

事件

事件名 事件参数 描述
$.parser.onComplete context 在解析器完成解析操作的时候触发。

 

方法

方法名 方法参数 描述
$.parser.parse context 解析EasyUI组件。
